# LOOKIA — Brandbook
> Guía de identidad visual. Versión 1.0 — Abril 2026

---

## 1. Identidad de marca

### ¿Qué es Lookia?
Lookia es una app de recomendación de outfits personalizada, dirigida a mujeres de 30+ con criterio propio que buscan una asesora de estilo inteligente, no una app juvenil ni básica.

### Posicionamiento
- **Sofisticada pero accesible** — cercana y práctica, no fría ni distante
- **Confiable** — las recomendaciones tienen fundamento real
- **Personalizada** — aprende del estilo real de la usuaria
- **Escalable** — B2C para usuarios finales, B2B para marcas y tiendas

### Personalidad de marca
| Atributo | Expresión |
|----------|-----------|
| Sofisticada | Tipografía serif, espacios en blanco generosos |
| Cálida | Paleta de tonos apagados y cremas |
| Inteligente | Copy directo, sin adornos innecesarios |
| Femenina | Acento rosado, proporciones delicadas |
| Premium | Dorado apagado para features avanzados |

## 2. Paleta de colores

### Colores principales

| Rol | Nombre | Hex | Uso |
|-----|--------|-----|-----|
| Base | Crema cálido | `#F5F0EB` | Fondo principal de la app |
| Texto | Carbón suave | `#2C2C2C` | Textos principales |
| Acento primario | Rosado palo | `#CD989D` | CTAs principales, highlights, logo |
| Acento oscuro | Rosado oscuro | `#A97074` | Hover states, énfasis |
| Neutro | Gris topo | `#9E9189` | Textos secundarios, bordes |
| Premium | Dorado apagado | `#C4A882` | Features B2B, planes premium, badges |

### Colores de soporte

| Rol | Nombre | Hex | Uso |
|-----|--------|-----|-----|
| Fondo secundario | Blanco cálido | `#FDFAF7` | Cards, modales |
| Borde suave | Gris claro cálido | `#E8E2DC` | Divisores, bordes de cards |
| Error | Tierra suave | `#C4796A` | Mensajes de error |
| Éxito | Verde salvia | `#8A9E8C` | Confirmaciones, estados positivos |

### Uso de color
- **Nunca** usar blanco puro `#FFFFFF` ni negro puro `#000000`
- El rosado `#CD989D` es el color firma — usarlo con moderación para que mantenga impacto
- El dorado `#C4A882` es exclusivo para contextos premium/B2B
- Los fondos deben ser siempre crema o blanco cálido, nunca gris frío

## 3. Tipografía

### Tipografías recomendadas

**Display / Logo**
- `Cormorant Garamond` — serif elegante, para títulos grandes y el logo
- Alternativa: `Playfair Display`

**Cuerpo / UI**
- `Inter` — sans-serif limpia y legible, para textos de interfaz
- Alternativa: `DM Sans`

**Acento / Editorial**
- `Cormorant Garamond Italic` — para citas, taglines, momentos editoriales

### Escala tipográfica

| Nivel | Tamaño | Peso | Fuente | Uso |
|-------|--------|------|--------|-----|
| H1 | 32px | 300 | Cormorant Garamond | Títulos de pantalla |
| H2 | 24px | 400 | Cormorant Garamond | Secciones principales |
| H3 | 18px | 500 | Inter | Subtítulos de cards |
| Body | 15px | 400 | Inter | Texto general |
| Caption | 12px | 400 | Inter | Etiquetas, metadata |
| Label | 11px | 500 | Inter uppercase | Tags, badges |

## 4. Logo

### Descripción
- La "L" mayúscula es un **colgador de ropa** estilizado en trazo caligráfico — comunica de inmediato el propósito de la app y funciona como símbolo de identidad único
- "ookia" en serif fino — equilibra la "L" caligráfica sin competir con ella
- Punto rosado `#CD989D` en la base del colgador — firma visual que ancla el logo y añade el único toque de color

### Variantes
| Variante | Uso |
|----------|-----|
| Logo completo sobre crema | Uso principal |
| Logo completo sobre oscuro | Fondos premium/B2B |
| Solo "L" caligráfica | Favicon, app icon |
| Logo en rosado | Casos especiales, campaigns |

### Espaciado
- Zona de protección: al menos el alto de la "L" en todos los lados
- Tamaño mínimo: 80px de ancho

## 6. Iconografía

### Estilo
- Línea fina (stroke 1.5px) — nunca filled
- Esquinas ligeramente redondeadas
- Tamaño base: 20x20px

### Set recomendado
- Lucide Icons — open source, estilo consistente con la marca
- Phosphor Icons — alternativa con variantes thin disponibles

## 7. Fotografía e imágenes

### Estilo
- Luz natural, tonos cálidos
- Fondos neutros — crema, blanco roto, gris claro
- Modelos 30+ o sin modelo (flat lay)
- Nunca imágenes con filtros fuertes o saturación alta

### Composición
- Espacios en blanco generosos
- Perspectiva limpia
- Paleta coherente con la marca

## 8. Tono de voz

### Principios
- **Directo** — sin rodeos, la usuaria sabe lo que quiere
- **Cálido pero no infantil** — como una amiga con buen gusto, no una influencer
- **Confiado** — las recomendaciones se presentan con seguridad
- **Honesto** — si algo no combina, se dice con tacto

### Ejemplos

| ❌ Evitar | ✅ Usar |
|-----------|---------|
| "¡Wow, qué look tan increíble!" | "Este outfit funciona bien para lo que tienes planeado" |
| "Ups, esto no combina para nada 😬" | "Esta prenda no es ideal para esta ocasión" |
| "¡Tu clóset es una locura!" | "Tienes buenas opciones para trabajar" |
| "Bestie, esto te va a encantar" | "Te recomendamos esto basado en tu estilo" |

---

## 9. Aplicación B2B

### Diferenciación visual
- El dorado `#C4A882` marca todo lo que es premium o B2B
- Tipografía más formal en materiales para marcas
- Logo sobre fondo oscuro `#2C2C2C` para presentaciones corporativas

### Materiales B2B
- Deck de ventas: fondo carbón `#2C2C2C`, acento dorado
- Propuestas: fondo crema, tipografía serif prominente
- Dashboard para marcas: misma paleta pero con más densidad de información
